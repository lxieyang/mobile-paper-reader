How to Get an API Key?

Step 1: Sign up at indico.io

This app relys on incido services to work. Therefore, you need an incido API key.

Go to indico.io/pay-per-call to sign up for a free account.

Putting in your info

Putting in your info

Step 2: Get an API Key

Incido allows 10000 free requests per month currently, which is more than enough for everyday uses. (Each time you read a pdf document, it counts as a request.)

Go to indico.io/dashboard to log in and get a new API key.

Generate your API key

Step 3: Set API key

You have just gained 10000 free accesses per month! Just one more step towards actually being able to use them!

Copy the API key from incido, go back to the Settings page to set your new API key.

Copy your new API key

Paste it in Settings

Hit Set API Key

Done!
